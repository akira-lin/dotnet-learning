## ⚡ 立即执行清单

```
Day 1：
□ 安装 Visual Studio 2022 + .NET 9 SDK
□ 运行第一个 Hello World 项目
□ 注册程序员客栈，看别人在接什么单

Day 3：
□ 完成 TodoList API（教程在 phase1-week3-4.md）
□ 把代码传到 GitHub（这是你的作品集）

Day 7：
□ 写一个爬虫脚本（采集公开数据）
□ 在程序员客栈投3个标（别怕被拒）

Week 2：
□ 完成第一个完整项目并部署
□ 开始在闲鱼/某宝挂服务
```

## 🎯 各阶段里程碑

| 阶段 | 时间 | 目标 | 能接的单 |
|------|------|------|---------|
| 阶段一完成 | 4个月 | 能独立完成 WebAPI 项目 | 小程序后端、简单后台、爬虫脚本 |
| 阶段二完成 | 8个月 | 能处理缓存、部署、异步 | 有一定复杂度的系统、微服务改造 |
| 阶段三完成 | 1年+ | 具备架构设计能力 | 整站外包、驻场开发、技术咨询 |
