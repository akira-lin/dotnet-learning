
## 📂 各周内容

### Week 1-2：.NET 基础 + 第一个 WebAPI

**目标：** 理解 .NET 9 开发流程，能写简单接口

**必须掌握的：**
- .NET 9 SDK 安装和项目创建
- ASP.NET Core WebAPI 项目结构
- HTTP 协议基础（GET/POST/PUT/DELETE）
- JSON 数据格式
- Postman 或 curl 测试接口
- 依赖注入（会用即可，不深究原理）

**详细教程：** [Week 1-2 教程](./phase1-week1-2.md)

---

### Week 3-4：数据库操作（EF Core + SQLite）

**目标：** 能用代码操作数据库，完成数据持久化

**必须掌握的：**
- EF Core 增删改查（Create/Read/Update/Delete）
- SQLite 数据库（轻量，无需安装数据库服务）
- DTO（数据传输对象）的使用
- 异步编程基础（async/await）
- 数据库迁移（Migration）

**详细教程：** [Week 3-4 教程](./phase1-week3-4.md)

---

### Week 5-8：接单技能包 + 数据处理脚本

**目标：** 掌握最容易变现的技术栈

**必须掌握的：**
- HttpClient 发请求
- HtmlAgilityPack 解析 HTML
- EPPlus / ClosedXML 操作 Excel
- JSON / CSV 文件处理
- 基础正则表达式
- 项目部署到 Windows Server / IIS

**详细教程：** [Week 5-8 教程](./phase1-week5-8.md)

## 📌 阶段一注意事项

1. **不要在某个点上卡太久**
   - IOC 容器原理、设计模式——这些以后再学
   - 先"能跑起来"比"跑得漂亮"重要

2. **每学一个功能就想怎么变现**
   - 学完 HttpClient → 想想能采集什么数据
   - 学完 EF Core → 想想什么场景需要数据库

3. **GitHub 是你的简历**
   - 每个项目都传上去
   - 写好 README 说明项目功能
   - 链接：https://github.com

4. **尽快接到第一单**
   - 不要等到"学完了"再接单
   - 第一单不图赚钱，图经验和评价
